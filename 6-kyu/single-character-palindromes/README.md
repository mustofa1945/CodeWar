<h2><a href=https://www.codewars.com/kata/5a2c22271f7f709eaa0005d3/train/javascript target="_blank">Single character palindromes</a></h2><h3>6 kyu</h3><p>You will be given a string and you task is to check if it is possible to convert that string into a palindrome by removing a single character. If the string is already a palindrome, return <code>"OK"</code>. If it is not, and we can convert it to a palindrome by removing one character, then return <code>"remove one"</code>, otherwise return <code>"not possible"</code>. The order of the characters should not be changed.</p><p>For example:</p><pre><code class="language-plaintext">"abba"   -&gt; "OK"           - this is a palindrome"abbaa"  -&gt; "remove one"   - remove the 'a' at the extreme right"abbaab" -&gt; "not possible" - cannot be made a palindrome </code></pre><p>Good luck!</p><p>If you like this Kata, please try <a href="https://www.codewars.com/kata/5a66ea69e6be38219f000110" data-turbolinks="false" target="_blank">Single Character Palindromes II</a></p>
